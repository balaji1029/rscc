# rscc

## Cloning the Repository

To clone the repository, use the following command:

```bash
git clone https://github.com/balaji1029/rscc.git
cd rscc
```

## Building the Project

To build the project, run:

```bash
cargo build
```

## Running the Project

To run the project, use:

```bash
cargo run -- <source_file>
```

Example source files can be found in the `examples` directory. For instance, to run `test1.lc`, use:

```bash
cargo run -- examples/test1.lc
```
